# 🍺 1708 — Система управления заказами бара

Real-time система для бара: приём заказов, управление очередью, учёт склада и гостевое меню по QR-коду.

---

## Структура проекта

## Роли

| Роль | Что видит |
|------|-----------|
| **Бармен** | Очередь заказов, кнопки «Начал» / «Готово» по каждой позиции |
| **Официант** | Новый заказ, список готовых к выдаче, кнопка «Отнёс» |
| **Менеджер** | Всё вышеперечисленное + статистика, редактор меню, управление столами |

---

## Основные функции

- **Real-time синхронизация** — заказы обновляются мгновенно на всех устройствах через Firebase
- **Гостевое меню** — гость сканирует QR, выбирает позиции и отправляет заказ
- **Учёт склада** — остатки списываются при заказе и возвращаются при удалении или редактировании
- **Мгновенная доставка** — пиво, напитки и закуски официант может сразу отметить как доставленные
- **Push-уведомления** — бармен получает звуковой сигнал и уведомление при новом заказе
- **Квиз** — генерация QR-кодов для всех столов для печати (режим квиз-бара)
- **PWA** — устанавливается на телефон как приложение

## Стек

- **Firebase Realtime Database** — хранение заказов, меню, столов
- **Firebase Auth** — анонимная авторизация
- **Vanilla JS (ES Modules)** — без фреймворков и сборщиков
- **PWA** — Service Worker, Web App Manifest
